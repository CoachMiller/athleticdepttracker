Privacy Policy

Athletic Dept Tracker respects your privacy and is committed to protecting your data. This application only accesses Google user data with explicit user authorization and uses that data solely to provide its intended functionality.

Data Accessed

Athletic Dept Tracker accesses the following Google user data only when authorized by the user:

  • Google Sheets data within the spreadsheet where the application is installed
  • Google Drive file access limited to the specific file in which the script is used
  • Basic Google account information (such as email address), as required for authorization

The application does not access any data outside of the files explicitly authorized by the user.

How Data Is Used

The data accessed is used strictly for the core functionality of the application, including:

  • Managing athlete roster data
  • Generating team and internal rosters
  • Calculating GPA and eligibility status
  • Producing reports and exports (including MaxPreps-ready formats)

All data processing occurs within the user’s Google Sheets environment.

Data Storage

  • All data remains within the user’s Google Sheets file
  • No data is copied, transferred, or stored outside of Google’s infrastructure
  • The application does not maintain its own database

Data Sharing

Athletic Dept Tracker does not sell, share, or transfer user data to any third parties.

  • No data is used for advertising purposes
  • No data is transmitted to external services
  • No data is accessed by anyone other than the authorized user

Data Security

  • All data access is handled through Google’s secure authentication system
  • Access is granted only after user consent
  • Permissions can be revoked at any time through the user’s Google account settings

User Control

Users maintain full control over their data at all times.

  • Revoke access via Google account permissions
  • Delete or modify data directly within Google Sheets
  • Stop using the application at any time
